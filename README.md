# Definance — Gestão Financeira Inteligente

O **Definance** é um sistema de gestão financeira pessoal que transforma a forma como você controla seu dinheiro.

A proposta é simples: você não precisa abrir um app ou preencher planilhas. Basta enviar uma mensagem — como se estivesse conversando — e o sistema cuida do resto.

🔗 Acesse a aplicação:
[https://definance.com.br](https://definance.com.br)

---

## Como funciona

Com o Definance, você pode:

* Registrar gastos e entradas apenas escrevendo uma mensagem
* Consultar seu saldo e extratos em tempo real
* Receber alertas inteligentes sobre seus hábitos financeiros
* Acompanhar tudo em dashboards claros e organizados

Tudo isso de forma automatizada, prática e acessível.

---

## Diferencial

O grande diferencial do Definance é a integração com inteligência artificial e comunicação direta via WhatsApp.

Você pode, por exemplo:

* “Gastei 50 reais com mercado”
* “Quanto eu gastei esse mês?”
* “Me mostra minhas contas a vencer”

E o sistema entende, processa e organiza essas informações para você.

---

## Principais funcionalidades

* Controle completo de receitas e despesas
* Categorias personalizáveis com limites de gasto
* Alertas configuráveis (gastos, vencimentos, atrasos, etc.)
* Dashboard com visão geral da vida financeira
* Histórico detalhado de transações
* Integração com IA para interpretação de mensagens
* Comunicação via WhatsApp
* Controle de contas fixas e variáveis
* Metas a serem conquistadas

---

## Tecnologias (visão geral)

O projeto é construído com uma arquitetura de microserviços, focada em escalabilidade, organização e evolução independente dos módulos, utilizando:

* Backend: .NET / C# / ASP.NET Core  
* Frontend: TypeScript  
* Banco de dados: PostgreSQL (via Supabase)  
* Módulo de IA: Python com FastAPI  
* Integração com IA: Groq API  
* Comunicação: integração via WhatsApp  

Detalhes técnicos mais aprofundados estão documentados separadamente em cada módulo do projeto.

---

## Objetivo

O Definance nasce com o objetivo de tornar a gestão financeira algo simples, acessível e inteligente — eliminando fricção e aproximando o controle financeiro do dia a dia real das pessoas.
